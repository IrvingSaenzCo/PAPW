Etiqueta: Section

Sirve para determinar qué contenido corresponde a qué parte de un esquema y representa una sección genérica de un contenido.

Etiqueta: Article

representa una composición auto-contenida en un documento, página, una aplicación o en el sitio, que se destina a distribuir de forma independiente o reutilizable

Etiqueta:nav

Sólamente contiene enlaces a las diferentes secciones de la web

Etiqueta: Blockquote

Se usa para escribir una cita textual o un párrafo exacto y que éste se diferencie del resto del texto. El efecto que nos muestra es que crea unos espacios a la derecha y a la izquierda del texto, para que se diferencie de los demás.

Etiqueta: ul

Hace referencia a hacer una lista de elementos no ordenada. Cada elemento debe ser agregado con una etiqueta hija "< li >< /li >".

Etiqueta: li

Representa un elemnto de una lista ordenada (etiqueta < ol >) o una lista no ordenada. El atributo value solamente está permitido en una lista ordenada, en una lista desordenada su valor en inválido.

Etiqueta: figure

Hace referencía a contenido relacionado, así como imagenes, ilustraciones.

Etiqueta: time

Es una etiqueta que define una manera de leer el tiempo de manera legible, este elemento también puede ser usado para codificar el tiempo de tal manera que la computadora lo pueda entender para que a través de esto se puedan ofrecer eventos dependiendo de la fecha, cumpleaños, etc..

Etiqueta: footer

Representa el pie de un documento o sección. La información que se suele añadir en este bloque es el autor del documento, enlaces a contenido relacionado, información de copyright, avisos legales, etc.

Etiqueta: mark

Esta etiqueta permite remarcar las palabras de un texto en este formato.

Etiqueta: aside

representa una sección de una página que tiene relación con el contenido que está mostrando la pagina web y que podría ser considerado por separado de ese contenido. Otro nombre al que se le puede hacer referencia es Barra lateral o menú lateral.
